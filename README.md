# global_reservation

## project description 
The Global Reservation System is a web-based platform designed to allow users to book and manage different types of reservations - such as flights , trains and buses in one unified system .

## Technologies used
HTML
CSS
JavaScript
PHP
MySQL

## Features
user login system 
Admin dashboard
Database connectivity

## How to run the project
Install XAMPP
Start Apache and MySQL
Place project folderinside htdocs
Import database using phpMyAdmin
Open browser and run localhost/project-folder-name

## Deployment
The project is hosted on a PHP-MySQL hosting server.
